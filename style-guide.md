# Visual Style

**Theme:** Dark navy developer UI  
**Font:** IBM Plex Sans (400/500/600/700) — [fonts.google.com](https://fonts.google.com/specimen/IBM+Plex+Sans), Apache 2.0

---

## Colours

| Token        | Value     | Use                        |
|--------------|-----------|----------------------------|
| Background   | `#080e1a` | Page background            |
| Surface      | `#0d1526` | Cards, panels              |
| Surface 2    | `#111d33` | Inputs, tags, nested items |
| Border       | `#1a2d4a` | All borders and dividers   |
| Accent       | `#1d6ff2` | Primary buttons, links, highlights |
| Accent hover | `#1558cc` | Hover state for accent     |
| Text         | `#dce8f5` | Body text                  |
| Text muted   | `#5e7a99` | Secondary text, labels     |
| Cyan         | `#22d3ee` | Secondary badge (desktop)  |
| Green        | `#22c55e` | Success / status           |

## Radii

| Element  | Value |
|----------|-------|
| Cards    | 6px   |
| Buttons  | 5px   |
| Badges   | 3–4px |

## Principles

- Dark navy (not pure black) — feels technical without being harsh
- Single blue accent, used sparingly — highlights only, not decoration
- Rectangular corners — structured and tool-like, not soft or playful
- IBM Plex Sans — formal enough for professional work, readable at small sizes
- Muted text at `#5e7a99` — high contrast only where it matters
- Subtle hover: `translateY(-2px)` + faint accent `box-shadow` on cards
